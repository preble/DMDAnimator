DMDAnimator Help

Using the Keyboard

The keyboard is the most efficient way to use DMDAnimator. Below are the keys recognized:

KeyPurpose
Arrow KeysMove over the DMD.
⇧-ArrowSelect a rectangular portion of the DMD.
⌘-ArrowShift/offset the contents of the DMD.
`Set the current pixel to be off (black).
1Set the current pixel to intensity level 1 (30%).
2Set the current pixel to intensity level 2 (60%).
3Set the current pixel to intensity level 3 (100%).
⇧⌘NCreate a new frame after the current frame.
, or ⌘[Step backward one frame.
. or ⌘]Step forward one frame.
SpaceToggle play/pause.

File Format

The format of .dmd files recognized by DMDAnimator is constructed as follows:

file() {
  number_of_frames                                1 byte
  for(i = 0; i < number_of_frames; i++) {
    frame()                                       4096 bytes
  }
}
frame() {
  for(row = 0; row < 32; row++) {
    for(col = 0; col < 128; col++) {
      pixel()                                     1 byte
    }
  }
}
pixel() {
  // 1 byte where value is:
  //   0 = off, 1 = low, 2 = medium, 3 = high intensity
  //   Clear state is not implemented.
}
